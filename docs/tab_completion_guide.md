# Tab 补全功能说明

在交互式 AI 聊天模式中，可以使用 Tab 键快速补全文件和目录路径。

## 功能特性

### @ 符号（文件引用）
- 输入 `@` 后按 Tab：显示当前目录的所有文件
- 输入 `@ 目录/` 后按 Tab：显示指定目录下的文件
- 输入 `@ 文件名部分` 后按 Tab：补全匹配的文件名

### # 符号（目录引用）
- 输入 `#` 后按 Tab：显示当前目录的所有子目录
- 输入 `# 目录名部分` 后按 Tab：补全匹配的目录名
- 目录补全会自动添加 `/` 后缀

## 使用示例

### 场景 1：查看当前目录的文件
```
你：@[Tab]
```
显示当前目录下的所有文件

### 场景 2：补全特定文件
```
你：@ REA[Tab]
```
自动补全为：
```
你：@ README.md
```

### 场景 3：查看子目录的文件
```
你：@ src/[Tab]
```
显示 `src/` 目录下的所有文件

### 场景 4：查看当前目录的子目录
```
你：#[Tab]
```
显示当前目录下的所有子目录

### 场景 5：补全目录名
```
你：# s[Tab]
```
自动补全为：
```
你：# src/
```

## 注意事项

1. **@ 模式**只显示文件，不显示目录
2. **# 模式**只显示目录，不显示文件
3. 如果有多个匹配项，再次按 Tab 会显示共同前缀
4. 如果只有一个匹配项，Tab 键会自动完成
5. 目录补全会自动添加 `/` 后缀

## 兼容性

- 支持跨平台路径（Windows、macOS、Linux）
- 支持大小写不敏感匹配
- 支持隐藏文件（以 `.` 开头的文件）

## 使用方式

1. 启动交互模式：
```bash
npm run dev -- ai
# 或
yuangs ai
```

2. 使用 Tab 补全：
```
你：@[Tab]  # 补全文件
你：#[Tab]  # 补全目录
```
