# Git 集成

yuangs 的智能 Git 功能。

## Commit

```bash
yuangs git commit              # 智能生成 commit message
yuangs git commit -a           # 暂存所有 + 提交
yuangs git commit --detailed   # 详细 commit message
yuangs git commit --type feat  # 指定类型
```

## Review

```bash
yuangs git review              # AI 代码审查
yuangs git plan                # 生成 todo.md
yuangs git exec                # 执行 todo.md 任务
yuangs git auto                # 自动执行
yuangs git resolve             # 冲突解决
```

## 其他

```bash
yuangs git status              # 增强状态显示
yuangs git branch              # 分支建议
yuangs git semdiff             # 语义 diff
yuangs git history             # 语义历史
```

详见 [git-workflow.md](git-workflow.md) 和 [git-shortcuts.md](git-shortcuts.md)。
